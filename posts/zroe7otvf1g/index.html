Ollama を使用し、ローカル環境で DeepSeek-R1 を実行する

Published: 2025-01-27
  • #AI
  • #Ollama
  • #DeepSeek-R1

概要

  • Ollama を使用し、ローカル環境で DeepSeek-R1 を実行する。

環境

  • ollama version is 0.5.7
  • NVIDIA GeForce RTX 3060 12GB

手順

Ollama のインストール

以下より、Ollama をダウンロードし、インストールする。

  • Ollama
ollama -v
ollama version is 0.5.7

モデルの実行

シェルを開き、DeepSeek-R1 を実行する。モデルは deepseek-r1 にて公開されている。パラメータ数に応じてモデルのサイズや快適に動作するスペックも変わるため、小さいモデルから試すのが良いかもしれない。

例えば、7b のモデルを使用する場合、以下コマンドを実行する。

ollama run deepseek-r1:7b

ダウンロードが終わると、プロンプトが表示される。

日本旅行を検討されているのであれば、京都や東京、大阪などの大都市だけでなく、自然豊かな地方にもぜひ足を運んでみてください！

いくつか質問を投げてみたが、明らかに精度が上がっている。ちょっとしたスクリプトも 7b だと動かないレベルだったが、32b だとそのままコピペで動くものになっている。

所感

ローカル環境で使うには、上位モデルを使う必要がある。その場合、それなりに高性能な GPU が必要となるため、個人レベルでの実用は難しそうである。しかし、OSS でこれだけのものが公開されることの影響は大きそう。
