Ubuntu 24.04 に Nextcloud をインストールする

Published: 2025-07-11
  • #Ubuntu

概要

  • Ubuntu 24.04 に Nextcloud をインストールする。
  • nginx をリバースプロキシとして使用し、HTTPS でアクセスする。

環境

  • Ubuntu 24.04 LTS
  • Nextcloud 31.0.6snap1
  • nginx 1.26.2
  • 独自ドメインを所有しており、証明書の発行が終わっていること。

手順

説明のためドメインは nextcloud.example.com を指定しているが、実際のドメインに置き換えて実施する。

Nextcloud のインストール

次のコマンドを実行し、Nextcloud をインストールする。

sudo snap install nextcloud

Nextcloud の設定

次のコマンドを実行し、HTTP のポート番号を 81 に変更する。

sudo snap set nextcloud ports.http=81

次のコマンドを実行し、HTTPS を無効化する。nginx 経由で接続するため、Nextcloud の HTTPS は無効化する。

sudo snap set nextcloud ports.https=""

次のコマンドを実行し、ドメインやプロトコルを設定する。

sudo nextcloud.occ config:system:set overwritehost --value="nextcloud.example.com"
sudo nextcloud.occ config:system:set overwriteprotocol --value="https"
sudo nextcloud.occ config:system:set overwritewebroot --value="/"
sudo nextcloud.occ config:system:set overwrite.cli.url --value="https://nextcloud.example.com"

次のコマンドを実行し、Nextcloud を再起動する。

sudo snap restart nextcloud

nginx の設定

任意のエディタで /etc/nginx/sites-available/nextcloud.example.com を作成する。

sudo vi /etc/nginx/sites-available/nextcloud.example.com
server {
    listen 443 ssl;
    listen [::]:443 ssl;
    server_name nextcloud.example.com;

    ssl_certificate /etc/letsencrypt/live/example.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/example.com/privkey.pem;

    charset utf-8;

    location / {
        proxy_pass_header Server;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        client_max_body_size 20G;
        proxy_pass http://localhost:81/;
    }

    error_page   500 502 503 504  /50x.html;
    location = /50x.html {
        root   /usr/share/nginx/html;
    }
}

次のコマンドを実行し、追加した設定を有効化する。

sudo ln -s /etc/nginx/sites-available/nextcloud.example.com /etc/nginx/sites-enabled/nextcloud.example.com

次のコマンドを実行し、nginx を再起動する。

sudo nginx -t | sudo systemctl reload nginx

Nextcloud の初期設定

ブラウザで https://nextcloud.example.com/ に接続する。

新しい管理者のアカウント名とパスワードを入力、インストール をクリックする。インストールが完了するまでしばらく待機する。

必要に応じて推奨アプリをインストールする。

ダッシュボードが表示されることを確認する。
