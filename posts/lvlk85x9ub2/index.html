C++ の std::map メモ

Published: 2022-10-26
  • #C++

概要

AtCoder で std::map を使うときのメモ書き。

基本

map

  • キーに設定した値の昇順にソートされる。
#include <bits/stdc++.h>
using namespace std;

int main() {
  map<int, int> mp;
  mp[5] = 10;
  mp[1] = 3;
  mp[4] = 6;
  for (auto p : mp) {
    printf("key:%d, value:%d\n", p.first, p.second);
  }
  return 0;
}
key:1, value:3
key:4, value:6
key:5, value:10

unorderd_map

  • ソートされないため、 map よりも高速に動作する。
#include <bits/stdc++.h>
using namespace std;

int main() {
  unordered_map<int, int> mp;
  mp[5] = 10;
  mp[1] = 3;
  mp[4] = 6;
  for (auto p : mp) {
    printf("key:%d, value:%d\n", p.first, p.second);
  }
  return 0;
}
key:4, value:6
key:1, value:3
key:5, value:10

キーの検索

find

  • find でキーを指定すると、該当する場合はイテレータを返す。
  • 該当しない場合は末尾のイテレータを返す。
#include <bits/stdc++.h>
using namespace std;

int main() {
  map<int, int> mp;
  mp[5] = 10;
  mp[1] = 3;
  mp[4] = 6;

  auto f = [](map<int, int> mp, int x) {
    auto it = mp.find(x);
    if (it == mp.end()) {
      cout << "該当なし。" << endl;
      return;
    }
    cout << it->first << ", " << it->second << endl;
  };

  // 対象キーが存在しない場合
  f(mp, 0);
  // 対象キーが存在する場合
  f(mp, 4);

  return 0;
}
該当なし。
4, 6

二分探索(lower_bound/upper_bound)

lower_bound

  • 指定したキー <= x となる最小のキー x を持つイテレータを返す。
  • 該当する x が存在しない場合、末尾のイテレータを返す。
#include <bits/stdc++.h>
using namespace std;

int main() {
  map<int, int> mp;
  mp[5] = 10;
  mp[1] = 3;
  mp[4] = 6;

  auto f = [](map<int, int> mp, int x) {
    auto it = mp.lower_bound(x);
    if (it == mp.end()) {
      cout << "該当なし。" << endl;
      return;
    }
    cout << it->first << ", " << it->second << endl;
  };

  f(mp, 3);
  f(mp, 4);
  f(mp, 5);
  f(mp, 6);

  return 0;
}
4, 6
4, 6
5, 10
該当なし。

upper_bound

  • 指定したキー < x となる最小のキー x を持つイテレータを返す。
  • 該当する x が存在しない場合、末尾のイテレータを返す。
#include <bits/stdc++.h>
using namespace std;

int main() {
  map<int, int> mp;
  mp[5] = 10;
  mp[1] = 3;
  mp[4] = 6;

  auto f = [](map<int, int> mp, int x) {
    auto it = mp.upper_bound(x);
    if (it == mp.end()) {
      cout << "該当なし。" << endl;
      return;
    }
    cout << it->first << ", " << it->second << endl;
  };

  f(mp, 3);
  f(mp, 4);
  f(mp, 5);
  f(mp, 6);

  return 0;
}
