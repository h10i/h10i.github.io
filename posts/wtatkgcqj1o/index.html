Windows のユーザやグループを PowerShell スクリプトで操作する

Published: 2024-09-11
  • #Windows
  • #PowerShell
  • #Windows 11
  • #PowerShell 5
  • #PowerShell 7

概要

  • Windows のユーザやグループをコマンドで操作する。
    • ユーザの作成/変更/削除
    • グループの作成/変更/削除
    • グループへのユーザ追加/削除

前提

  • PowerShell 5.1
    • PowerShell 7+ の場合、モジュールインポートが必要である。
    • 各コマンドは、管理者権限の PowerShell で実行する。

説明

PowerShell 7+ の場合

PowerShell 7+ で作成する場合、New-LocalUser を実行すると、次のようなエラーが出る。

New-LocalUser: Could not load type 'Microsoft.PowerShell.Telemetry.Internal.TelemetryAPI' from assembly 'System.Management.Automation, Version=7.4.5.500, Culture=neutral, PublicKeyToken=31bf3856ad364e35'.

PowerShell の Issue として報告されている。

  • Running New-LocalUser in PowerShell 7.3 raises an error · Issue #18624 · PowerShell/PowerShell
  • New-LocalUser doesn’t work in Power 7.3+ without importing Microsoft.Powershell.LocalAccounts · Issue #21645 · PowerShell/PowerShell

以下のコマンドでモジュールをインポートすると、問題は解消する。今回の手順では、以下をインポートした状態を前提として説明する。

Import-Module Microsoft.PowerShell.LocalAccounts -UseWindowsPowerShell

また、issue のコメントにて、以下のモジュールをインストールすれば、 PowerShell 7+ で使えるとのことである。(試してないため、今回記載するコマンドが問題なく実行できるかは不明である。)

  • iSazonov/LocalAccounts： vNext version of Microsoft.PowerShell.LocalAccounts for .Net 8+

ユーザ/グループの参照

ユーザの参照

ユーザ情報は、以下のコマンドで参照できる。

  • Get-LocalUser (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

全てのユーザ情報を参照する場合、引数なしで Get-LocalUser コマンドを実行する。

Get-LocalUser

特定ユーザの情報のみを参照する場合、-Name にユーザ名を指定する。

$userName = "TestUser"
Get-LocalUser -Name $userName

グループの参照

グループ情報は、以下のコマンドで参照できる。

  • Get-LocalGroup (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

全てのグループ情報を参照する場合、引数なしで Get-LocalGroup コマンドを実行する。

Get-LocalGroup

特定グループの情報のみを参照する場合、-Name にグループ名を指定する。

$groupName = "TestGroup"
Get-LocalGroup -Name $groupName

グループメンバの参照

グループに所属するメンバの情報は、以下のコマンドで参照できる。

  • Get-LocalGroupMember (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestGroup グループの所属ユーザ全てを参照するには、以下の Get-LocalGroupMember コマンドを実行する。

$groupName = "TestGroup"
Get-LocalGroupMember -Name $groupName

TestGroup グループ内の TestUser ユーザのみの情報を参照するには、-Member オプションに対象ユーザ名を指定する。

$groupName = "TestGroup"
$userName = "TestUser"
Get-LocalGroupMember -Name $groupName -Member $userName

全てのユーザと所属グループをまとめて参照

以下のスクリプトを実行することで、全ユーザ情報と各ユーザが所属するグループをまとめて出力できる。

$localUsers = Get-LocalUser

$result = @()

foreach ($user in $localUsers) {
  $groups = Get-LocalGroup | ForEach-Object {
    $groupName = $_.Name
    if (Get-LocalGroupMember $groupName -Member $user.Name -ErrorAction SilentlyContinue) {
        $groupName
    }
  }

  $result += [pscustomobject]@{
    UserName = $user.Name
    Enabled = $user.Enabled
    Groups = $groups -join ", "
    UserDescription = $user.Description
  }
}

$result | Format-Table -AutoSize

ユーザの操作

ユーザの作成

Windows の新しいローカルユーザは、以下のコマンドで作成できる。

  • New-LocalUser (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

今回は、ユーザ名とパスワードのみを設定したローカルユーザを作成する。

例えば、ユーザ名が TestUser、パスワードが TestUserPassword のユーザを作成するには、以下コマンドを実行する。

$userName = "TestUser"
$plainPassword = "TestUserPassword"
$securePassword = ConvertTo-SecureString -String $plainPassword -AsPlainText -Force

New-LocalUser -Name $userName -Password $securePassword

ユーザ名とパスワードを対話形式で入力する場合、以下のようにする。

$userName = Read-Host "Please enter the new user name and press Enter"
$securePassword = Read-Host -AsSecureString "Please enter the new password for '$userName' and press Enter"

New-LocalUser -Name $userName -Password $securePassword

ユーザの変更

ユーザ名の変更

ローカルユーザのユーザ名は、以下のコマンドで変更できる。

  • New-LocalUser (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestUser ユーザの名前を NewTestUser に変更する場合、以下コマンドを実行する。

Rename-LocalUser -Name "TestUser" -NewName "NewTestUser"
パスワードの変更

ローカルユーザのパスワードは、以下のコマンドで変更できる。パスワード以外も変更できるが、今回はパスワードのみの手順を記載する。

  • Set-LocalUser (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、プロンプトで入力したユーザのパスワードを変更するには、以下のコマンドを実行する。

$userName = Read-Host "Please enter the user name and press Enter"
$secureNewPassword = Read-Host -AsSecureString "Please enter the new password for '$userName' and press Enter"
$userAccount = Get-LocalUser -Name $userName
$userAccount | Set-LocalUser -Password $secureNewPassword

ユーザの削除

ローカルユーザは、以下のコマンドで削除できる。

  • Remove-LocalUser (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestUser ユーザを削除するには、以下の Remove-LocalUser コマンドを実行する。

$userName = "TestUser"
Remove-LocalUser -Name $userName

グループの操作

グループへのユーザ追加

グループには、以下のコマンドでユーザを追加できる。

  • Add-LocalGroupMember (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestUser ユーザを Administrators グループに追加するには、以下の Add-LocalGroupMember コマンドを実行する。

$group = "Administrators"
$userName = "TestUser"
Add-LocalGroupMember -Group $group -Member $userName

グループからユーザ削除

グループからは、以下のコマンドでユーザを削除できる。

  • Remove-LocalGroupMember (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestUser ユーザを Administrators グループから削除するには、以下の Remove-LocalGroupMember コマンドを実行する。

$group = "Administrators"
$userName = "TestUser"
Remove-LocalGroupMember -Group $group -Member $userName

グループの作成

Windows の新しいローカルグループは、以下のコマンドで作成できる。

  • New-LocalGroup (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestGroup を作成するには、以下の New-LocalGroup コマンドを実行する。

$groupName = "TestGroup"
New-LocalGroup -Name $groupName

グループの変更

グループ名の変更

ローカルグループのグループ名は、以下のコマンドで変更できる。

  • Rename-LocalGroup (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestGroup グループの名前を NewTestGroup に変更するには、以下の Rename-LocalGroup コマンドを実行する。

$groupName = "TestGroup"
$newGroupName = "NewTestGroup"
Rename-LocalGroup -Name $groupName -NewName $newGroupName
グループ説明文の変更

ローカルグループのグループ説明文は、以下のコマンドで変更できる。

  • Set-LocalGroup (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestUserGroup グループの説明文を This is a test sample description. に変更するには、以下の Set-LocalGroup コマンドを実行する。

$groupName = "TestGroup"
$description = "This is a test sample description."
Set-LocalGroup -Name $groupName -Description $description

グループの削除

ローカルグループは、以下のコマンドで削除できる。

  • Remove-LocalGroup (Microsoft.PowerShell.LocalAccounts) - PowerShell ｜ Microsoft Learn

例えば、TestGroup グループを削除するには、以下の Remove-LocalGroup コマンドを実行する。

$groupName = "TestGroup"
Remove-LocalGroup -Name $groupName
