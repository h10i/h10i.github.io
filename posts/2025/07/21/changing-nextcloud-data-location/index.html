Nextcloud のデータ保存先を変更する

概要#

  • Nextcloud のデータ保存先を外付け HDD に変更する。

環境#

  • Ubuntu 24.04 LTS
  • Nextcloud 31.0.6snap1

手順#

外付け HDD は /media/hdd にマウントされているものとする。

保存先の作成#

/mnt または /media 配下に新たな保存先を作成する。

sudo mkdir -p /media/hdd/nextcloud
sudo chown -R root:root /media/hdd/nextcloud

Nextcloud の設定変更#

Nextcloud に外付けメディアへのアクセス権限を設定する。

sudo snap connect nextcloud:removable-media

Nextcloud を停止する。

sudo snap stop nextcloud

任意のエディタで Nextcloud の設定ファイルを開く。

sudo vi /var/snap/nextcloud/current/nextcloud/config/config.php

datadirectory のパスを変更し、上書き保存する。

 // ...
 'datadirectory' => '/media/hdd/nextcloud/data',
 // ...

既存データがある場合は移行する。

sudo mv /var/snap/nextcloud/common/nextcloud/data /media/hdd/nextcloud/

Nextcloud を開始する。

sudo snap start nextcloud
