Windows 11 (WSL) に Amazon Q CLI をインストールする

2025-10-24 · 3 min · Me | Suggest Changes
Table of Contents
  • 概要
  • 環境
  • 手順
  • 参考

概要#

  • Amazon Q CLI は Windows に対応していないため、WSL 上の Ubuntu にインストールする。
  • 無料枠で試すため、AWS Builder ID を使用する。

環境#

  • Windows 11 (WSL/Ubuntu 24.04)

手順#

Ubuntu (WSL) にて実行する。

amazon-q のパッケージをダウンロードする。

wget https://desktop-release.q.us-east-1.amazonaws.com/latest/amazon-q.deb

ダウンロードしたパッケージをインストールする。

sudo dpkg -i amazon-q.deb
sudo apt-get install -f

q コマンドを実行する。

q

error: You are not logged in, please log in with q login

ログインしろと言われるため、指示通りにする。

q login

? Select login method ›
❯ Use for Free with Builder ID
  Use with Pro licens

ログインするアカウントを選択する。今回はお試しなので、AWS Builder を使用する Use for Free with Builder ID を選択する。

✔ Select login method · Use for Free with Builder ID

Confirm the following code in the browser
Code: XXXX-XXXX

Open this URL: https://view.awsapps.com/start/#/device?user_code=XXXX-XXXX
▰▰▰▰▰▱▱ Logging in...

表示された URL を開き、認証する。コンソールに以下が表示されたら認証に成功している。

Device authorized
Logged in successfully

認証に成功したら、Amazon Q を起動する。

q
