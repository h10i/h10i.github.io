概要#

  • Detekt Gradle Plugin を使用して、detekt を実行する。
  • Git pre-commit hook を使用して、コミット時に detekt を自動実行する。

環境#

  • Gradle 8.13
  • detekt 1.23.8
  • detekt-formatting 1.23.8

手順#

Gradle Plugin の設定#

build.gradle.kts に detekt を追加する。(detekt のドキュメントには io.gitlab.arturbosch.detekt が記載されているが、2.0.0-alpha.0 以降を使う場合は dev.detekt とする必要がある。)

plugins {
    id("io.gitlab.arturbosch.detekt") version "[version]"
}

自動フォーマットさせたい場合、detekt-formatting を追加する。

dependencies {
    detektPlugins("io.gitlab.arturbosch.detekt:detekt-formatting:[version]")
}

detekt の Config を生成する。

gradlew detektGenerateConfig

config/detekt/detekt.yml が作成される。

build:
  maxIssues: 0
  excludeCorrectable: false
  weights:
    # complexity: 2
    # LongParameterList: 1
    # style: 1
    # comments: 1

config:
  validation: true
  warningsAsErrors: false
  checkExhaustiveness: false
  # when writing own rules with new properties, exclude the property path e.g.: 'my_rule_set,.*>.*>[my_property]'
  excludes: ""
# ...

build.gradle.kts に以下を追加する。

detekt {
    config.setFrom(file("config/detekt/detekt.yml"))
    buildUponDefaultConfig = true
    autoCorrect = true
}

detekt を実行する。

gradlew detekt

ルールに反するコードがある場合、該当箇所と抵触したルールが表示される。また、detekt.yml の formatting にて設定しているルールは自動フォーマットされる。

Git pre-commit の設定#

.git/githooks/pre-commit を作成し、以下を記載する。

#!/usr/bin/env bash
echo "Running detekt check..."
OUTPUT="/tmp/detekt-$(date +%s)"
./gradlew detekt > $OUTPUT
EXIT_CODE=$?
if [ $EXIT_CODE -ne 0 ]; then
  cat $OUTPUT
  rm $OUTPUT
  echo "***********************************************"
  echo "                 detekt failed                 "
  echo " Please fix the above issues before committing "
  echo "***********************************************"
  exit $EXIT_CODE
fi
rm $OUTPUT

この状態でコミットすると detekt が実行される。detekt のチェックが通ればコミットされ、失敗するとコミットされない。

チームで共有する場合は、共有用のディレクトリを作成し、その配下に pre-commit を配置する。

mkdir .githooks
tee .githooks/pre-commit<<EOF
#!/usr/bin/env bash
echo "Running detekt check..."
OUTPUT="/tmp/detekt-$(date +%s)"
./gradlew detekt > $OUTPUT
EXIT_CODE=$?
if [ $EXIT_CODE -ne 0 ]; then
  cat $OUTPUT
  rm $OUTPUT
  echo "***********************************************"
  echo "                 detekt failed                 "
  echo " Please fix the above issues before committing "
  echo "***********************************************"
  exit $EXIT_CODE
fi
rm $OUTPUT
EOF

実行権限を付与する。

chmod a+x .githooks/*

Git hooks のパスを変更する。

git config --local core.hooksPath .githooks

.githooks をプロジェクトの管理対象としておけば、Git hooks のパス変更のみで pre-commit の設定を反映できる。
