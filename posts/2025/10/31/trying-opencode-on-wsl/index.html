概要#

  • OpenCode とは、任意の LLM プロバイダを選べる OSS の AI エージェントである。
  • Claude Code や Gemini CLI のように、ターミナル上で動作する。

環境#

  • WSL (Ubuntu 24.04 LTS)
  • opencode 0.15.30

手順#

インストール#

OpenCode を npm でインストールする。

npm i -g opencode-ai@latest

OpenCode のバージョンを確認する。

opencode -version

0.15.30

次のコマンドを実行し、プロバイダと API キーを入力する。

選択肢にないプロバイダを設定したい場合は、設定ファイルに記載する。

例えば、ローカルで動かしている ollama をプロバイダとして使用する場合は、次の設定を設定ファイルに記載する。

vi ~/.config/opencode/opencode.json

{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "ollama": {
      "npm": "@ai-sdk/openai-compatible",
      "options": {
        "baseURL": "http://localhost:11434/v1"
      },
      "models": {
        "gpt-oss:20b": {
          "name": "gpt-oss:20b"
        }
      }
    }
  }
}

ollama は認証を必要としないためこのまま使える。認証が必要なプロバイダを追加した場合は opencode auth login から Other を選択し、登録した provider ID を入力する。

動作確認#

OpenCode を起動する。

opencode

モデルを選択する。

/models

指示を出してみる。

C言語で Hello World を出力するファイルを作成してください。
h10i (08:39 PM)

C言語で "Hello World" を出力するプログラムを hello.c というファイル名で作成します。
Build gemini-flash-latest (08:39 PM)

Write hello.c

#include <stdio.h>

int main() {
    printf("Hello World\n");
    return 0;
}

実際にファイルが作成されている。

ls

hello.c

opencode を終了する。

/exit

コンパイルして実行してみる。

gcc -o hello hello.c
./hello

Hello World

opencode run コマンドを使うと、プログラム的に動かせる。

opencode run "このコードの内容を説明して" < hello.c

このプログラムは、C 言語で書かれた「Hello World」を画面に表示する最も基本的な例です。行ごとの意味は以下の通りです。

```c
#include <stdio.h>
```

- 標準入出力ライブラリをインクルードします。`printf` 関数はこのヘッダーで定義されているので、文字列を画面に出力できるようになります。

```c
int main() {
```

- プログラムのエントリーポイント（入口）となる `main` 関数です。戻り値の型は `int` で、実行結果を OS に返すために使われます。

```c
    printf("Hello World\n");
```

- `printf` 関数を呼び出し、文字列 **Hello World** と改行文字 `\n` を標準出力（通常はコンソール）に表示します。

```c
    return 0;
```

- `main` 関数から整数 `0` を返します。`0` は「正常終了」を意味し、OS に対してプログラムがエラーなく終わったことを伝えます。

```c
}
```

- `main` 関数の終端です。

### 実行手順（参考）

1. ソースコードを `hello.c` のようなファイルに保存します。
2. コンパイル：`gcc hello.c -o hello`（または `clang` でも同様）
3. 実行：`./hello`
   → 画面に `Hello World` と表示されます。

要するに、このプログラムは「Hello World」を画面に出力し、正常に終了するだけの最小限の C プログラムです。
